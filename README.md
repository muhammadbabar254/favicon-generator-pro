# 🎨 Favicon Generator Pro

A modern favicon generator built with HTML, CSS, and JavaScript. Upload images, generate multiple favicon sizes, preview results, and download optimized favicon assets instantly.

---

## 🚀 Features

### 📤 Image Upload

- Upload logo or image
- Drag & Drop support
- Instant image preview

### 🎯 Favicon Generation

- Generate 16×16 favicon
- Generate 32×32 favicon
- Generate 64×64 favicon
- Real-time favicon preview

### 📥 Download

- Download generated favicon
- PNG format support
- One-click export

### 🖥 Modern Interface

- Responsive Design
- Dark Theme UI
- Clean Dashboard Layout

## ⚙️ How It Works

### Step 1

Upload your logo or image.

### Step 2

Click "Generate Favicons".

### Step 3

Preview generated favicon sizes.

### Step 4

Download the generated favicon.
